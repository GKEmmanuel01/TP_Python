# Travaux Pratiques de Python

## Présentation
Ce dépôt contient une série de **travaux pratiques (TP)** réalisés dans le cadre de l’apprentissage du langage **Python**.  
Les TP couvrent les bases de la programmation, la programmation orientée objet, la manipulation de fichiers, les algorithmes et le traitement de données.

Chaque TP est contenu dans un fichier séparé afin de garantir une bonne lisibilité et une meilleure organisation du code.

---

## Contenu du projet

| TP | Thème |
|----|------|
| TP 1 | Prise en main, variables, conditions et boucles |
| TP 2 | Fonctions et modularité |
| TP 3 | Listes, tuples et dictionnaires |
| TP 4 | Manipulation des chaînes de caractères |
| TP 5 | Lecture et écriture de fichiers |
| TP 6 | Gestion des exceptions |
| TP 7 | Programmation Orientée Objet (POO) |
| TP 8 | Algorithmes et complexité |
| TP 9 | Manipulation de données CSV |
